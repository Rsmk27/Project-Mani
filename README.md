# 🧠 Mani — AI Assistant by RSMK Technologies

Mani is a modular AI assistant system developed by **RSMK Technologies**, designed to function as a centralized intelligence layer across multiple web applications.

Unlike traditional chatbots, Mani follows an **orchestrator-based architecture**, enabling it to route user queries intelligently between different systems such as AI models, local logic, and contextual knowledge sources.

---

## 🚀 Vision

Mani is built to become the **core AI interface** across all RSMK platforms, including:

* Portfolio systems
* BudgetBuddy (finance assistant)
* GridForge (smart energy simulations)
* Future intelligent applications

The goal is to create a **unified AI experience** that understands context, adapts to domains, and maintains a consistent identity.

---

## 🏗️ Architecture Overview

Mani is inspired by modern assistant systems like Siri and Google Assistant.

### Core Layers:

* **Mani Core (Orchestrator)**
  Handles user input, intent detection, and routing decisions

* **AI Model Layer**
  Integrates external models via Hugging Face

* **Tooling Layer**
  Handles tasks like calculations and logic

* **RAG Layer (Planned)**
  Enables website-specific knowledge retrieval

* **Response Builder**
  Ensures consistent tone and identity
